
# 프로젝트 개요

이 프로젝트는 대한민국에서 일하거나 일하기를 희망하는 베트남 사람들을 위한 한국어 학습용 웹 애플리케이션입니다. 유튜브 영상을 활용하여 실제적인 한국어 표현을 학습하고, 반복 학습 및 영상 제어 기능을 통해 학습 효율을 높이는 것을 목표로 합니다.

# 주요 기능

- **유튜브 영상 임베드**: 한국어 학습 콘텐츠가 포함된 유튜브 영상을 웹앱에 임베드합니다.
- **영상 제어**: 되돌려 보기, 반복 재생, 재생 속도 조절 등 학습에 필요한 영상 제어 기능을 제공합니다.
- **타임스탬프 문장 카드**: 영상에 등장하는 주요 한국어 표현을 문장 카드 형태로 제공하며, 각 카드를 클릭하면 영상의 해당 표현이 나오는 위치로 즉시 이동합니다.
- **AI 기반 번역 및 표현 생성 (추후 확장)**: GenAI를 활용하여 한국어 및 베트남어 번역, 추가 학습 표현 생성 기능을 통합할 예정입니다.

# 디자인 및 스타일

- **모던하고 직관적인 UI**: 사용자가 쉽게 기능을 이해하고 사용할 수 있도록 깔끔하고 현대적인 디자인을 적용합니다.
- **반응형 웹 디자인**: 데스크톱, 태블릿, 모바일 등 다양한 디바이스에서 최적의 학습 경험을 제공합니다.
- **시각적 요소 활용**: 아이콘, 색상, 애니메이션 등을 활용하여 학습의 재미와 몰입도를 높입니다.

# 현재 구현 계획

1.  **`index.html`**:
    *   앱의 기본 구조(헤더, 영상 플레이어 영역, 컨트롤 버튼, 문장 카드 리스트)를 설정합니다.
    *   제목을 "베트남인을 위한 한국어 학습"으로 변경합니다.
2.  **`style.css`**:
    *   `index.html`의 각 요소에 대한 기본 스타일을 적용하여 시각적으로 보기 좋게 만듭니다.
    *   헤더, 영상 플레이어, 컨트롤 버튼, 문장 카드에 대한 스타일을 정의합니다.
3.  **`main.js`**:
    *   유튜브 영상 ID를 임시로 설정합니다.
    *   되돌려 보기, 반복, 재생 속도 조절 등 영상 제어 기능을 구현합니다.
    *   문장 카드 클릭 시 해당 영상 시간으로 이동하는 기능을 구현합니다.
